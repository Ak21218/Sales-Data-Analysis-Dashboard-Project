# Superstore Sales Data Analysis and Dashboard

This project analyzes the `SampleSuperstore.csv` dataset using Python to uncover sales and profit patterns across regions and product categories. It also includes dashboard assets for presenting the findings visually.

## Objective
Analyze sales data and extract meaningful business insights such as revenue trends, regional performance, and category-level analysis to support data-driven decision making.

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Power BI

## Dataset
- File: `SampleSuperstore.csv`
- Records: 9,994
- Columns: 13
- Key fields: `Region`, `Category`, `Sales`, `Profit`, `Discount`, `Quantity`

## What the Analysis Covers
- Dataset preview and structural inspection
- Column and data type review
- Missing value checks
- Descriptive statistics for numeric columns
- Total sales and total profit calculation
- Sales aggregation by region
- Sales aggregation by category

## Key Business Insights
- Total Sales: $2,297,200.86
- Total Profit: $286,397.02
- West region generated the highest sales at $725,457.82
- Technology was the top-performing category with $836,154.03 in sales
- South region contributed the lowest sales, indicating room for growth
